# Описание

Представьте что вам понадобилось взять только различные значения из переменной, данные в которую может вносить пользователь в визуальной части.

Пользователь вводит значения: "1 1 1 1 2 2 2...", а как нам из этого списка оставить только уникальные записи?

Это может пригодится например при построении ODAG приложений, или для визуального анализа введенных данных или для других случаев

<!-- truncate -->

В моём примере, переменная называется `myVar`, разделитель - пробелы

![1](/img/blog/2024/08-09/Rh3qKlzGE5.png)

Формула очень простая, забираем и пользуемся:
```
=concat(distinct SubField('$(myVar)',' ',ValueLoop(1,$(=SubStringCount(myVar,' '))+1)),' ')

```
