# Скрытые возможности

## Открытие 2 вкладок с разными фильтрами в одном приложении

Например, нам нужно в одном приложении сделать 2 разных набора фильтров и визуально сравнить результаты выборок

Можно прибегнуть к скрытой возможности идентификации нашей вкладки в браузере.

Делается это путём добавления к браузерной строке параметра со значением,
например у нас в URL ссылка такого вида: `https://qlik.sense.domain.com/app/guid/view/sheetid/`

добавляем к этому адресу параметр: `identity/1`, чтобы получится вот такой URL : `https://qlik.sense.domain.com/app/guid/view/sheetid/identity/1`

таким образом мы передаем в Qlik Engine информацию, о том, что мы используем уникальный идентификатор работы с нашим приложением

Для второй вкладки, повторяем тоже самое, только используем другую цифру, например - 2, итоговый URL будет следующим: `https://qlik.sense.domain.com/app/guid/view/sheetid/identity/2`

Вот и всё, теперь у 2 незавимые вкладки одного приложения, и выборки в них - не пересекаются
